# Co-Reply (コー・リプライ)
**AIと人間のシナジーによる次世代SNSエンゲージメント・アシスタント**

Co-Replyは、SNSでの「探す・考える・書く」というプロセスを劇的に効率化する、ハイブリッド型マーケティングプラットフォームです。AIによる高品質な下書き作成と、人間による最終確認を組み合わせることで、アカウントの安全性を守りつつ、本物のファン作りをサポートします。

---

## 🌟 主な機能

### 1. ターゲット・スカウト (データ収集)
指定したキーワードやハッシュタグに基づき、X (Twitter) から投稿を自動で収集します。
- **マルチ言語対応:** 日本語・英語の検索クエリをサポート。
- **スマート・ストレージ:** 重複を排除し、Supabaseへ効率的にデータを保存。
- **レート制限への配慮:** クエリ頻度や取得ボリュームを柔軟に設定可能。

### 2. AIドラフト・エンジン (LLM連携)
Google Gemini APIを活用し、文脈に応じたパーソナライズされた返信案を生成します。
- **ペルソナ注入:** ブランドの性格や目的（リード獲得、コミュニティ作り、サポート等）を定義。
- **言語自動検出:** 投稿の言語を自動で判別し、適切な言語で返答を作成。
- **クオリティ制御:** 共感的で押し付けがましくない、人間らしいトーンに最適化。

### 3. 洗練されたダッシュボード (UX)
ミニマルで直感的なインターフェースで、返信の確認・編集・送信を効率的に行えます。
- **洗練されたデザイン:** ダークモード優先のモダンなUI、高品質なタイポグラフィを採用。
- **ステータス管理:** 「保留」「承認（送信済み）」「スキップ」などの状態を可視化。
- **ワンクリック連携:** X Web Intentを利用し、ブラウザから直接、安全に送信が可能。
- **レスポンシブ対応:** デスクトップはもちろん、モバイルでの作業も快適に。

---

## 🛠 技術スタック

- **Framework:** Next.js 15+ (App Router)
- **Backend-as-a-Service:** Supabase (PostgreSQL, Auth, Edge Functions)
- **AI Provider:** Google Gemini API (Google AI SDK)
- **Styling:** Tailwind CSS + shadcn/ui
- **Icons:** Lucide-react
- **Localization:** Next.js Internationalization / i18next

## 🚀 開発を始める

### 1. 依存関係のインストール
```bash
npm install
```

### 2. 環境変数の設定
`.env.local` ファイルを作成し、必要なAPIキーを設定してください。
```env
NEXT_PUBLIC_SUPABASE_URL=your_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
GEMINI_API_KEY=your_key
```

### 3. 開発サーバーの起動
```bash
npm run dev
```
ブラウザで [http://localhost:3000](http://localhost:3000) を開いて確認できます。

## 📄 デザイン哲学
Co-Replyは「Minimalist & Dark-first」の美学に基づいています。無駄を削ぎ落としたインターフェースは、ユーザーが余計なノイズに惑わされることなく、純粋にコミュニケーションに集中できる環境を提供します。
